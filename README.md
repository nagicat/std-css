# std.css
std.cssは、クラス名の指定を必要としないCSSフレームワークです。

[詳細](https://nagicat.github.io/std-css/)

## Quick start
```html
<!DOCTYPE html>
<html>
<head>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/nagicat/std-css/std.min.css">
    <title>Document</title>
</head>
<body>
    <h1>Hello world!</h1>
</body>
</html>
```
